# 股票投资入门

## 能力圈
- 巴菲特，价值投资，760亿， 20%年投资回报率
- 判断准确度高，区别投资和投机，避免重大损失
- 错误做法
    + 投资热门股票
    + 不做产盘和服务研究
    + 购买别人推荐的股票
    + 跟风购买别人挣钱的股票
- 对产品服务熟悉，对公司有调查，对自己的投资有信心(70%)
***

## 安全边际
- 安全边际是价值和价格的容错空间，不要加杠杆！
- 减少损失，扩大受益
- 确定性和安全边际成比例
    + 确定性越低，需要留出更大的安全边际
- 出现时机：市场，行业，公司危机
- 常见错误：没有对比价值和价格, 斗鱼IPO投资失败案例
#### 明确投资目标
- 长期投资目标
- 个股投资目标, 年回报率
- 目标设定
    + 快，小，市场波动
    + 长，大，公司价值提升或者低估
#### 参与者
- 华尔街
    + 倾向看涨
- 机构投资者
    + 考核机制，相对回报（相对同行的盈利状况）
    + 投资限制
- 监管部门
    + 关注下跌
#### 价格的决定因素
+ 公司的盈利能力：长期平均价格
+ 供求关系：短期价格
#### 催化剂
- 公司盈利回归
- 盈利增长
- 新业务的商业模式
- 短期催化剂：比如公司年度产品
- 长期催化剂：模式或者盈利能力，比如云计算
#### 能力圈 vs 安全边际
- 能力圈：投资对的公司
- 安全边际：投资对的价格
***

## 护城河
- 保护自己未来盈利能力的积累，相对竞争优势的累积
    + 针对现有竞争者
    + 针对潜在竞争者
- 类型
    + 专利型：芯片，制药
    + 品牌：可口可乐，穆迪
    + 技术：台积电，英伟达
    + 经营效率型：costco，小米
    + 网路效应：脸书
- 深度，相对竞争优势
    + 公司本身的积累
    + 竞争对手的积累
- 商业模式
    + 提供价值
    + 获利方式
- 概率
    + 增加判断正确的概率
    + 减低判断错误的概率

## 财务报表
- 损益表：分析公司盈利能力
- 资产负债表：分析同事财务状况，财务风险
- 现金流量表：公司流动性能力
#### 报表查询
- sec.gov
#### 财务报表
- 倾向于持久和稳定的利润
- 避免周期型公司，不确定因素，不推荐非行业专业人士
- 财务数据： gurufocus.com， 查看10年以上的财务数据
#### 优点和缺点
- 标准化，信息透明
- 数量分析为主，质量取决于经营模式，管理，行业，风险
#### 损益表
- 市场定价来源
- 衡量公司价值
    + e.g. 糖果店
    + 营业额： 提价能力，销量，产品线
    + 产品成本, 毛利率: 产品成本，原材料和制作工艺
        + 控制成本，供应链地位，原材料生成
        + 价值溢价，专利和技术
        + 品牌和认可
        + 对手退出，形成寡头：比如数字货币交易所
    + 运营成本: 房租，人工，广告，折旧
        + 销售和一般费用
            + 营销成本升高说明竞争力下降，营销指向渠道和品牌
            + 工资过高说明结构问题
        + 研发费用
            + 非科技公司： 预期比较低
            + 科技公司：足够投入，长期积累
        + 折旧
            + 不能占营业额太多
    + 特别项目
        + 出售资产：聚焦优势项目或者回收现金流，反向推论买入的资产是否恰当 
        + 商誉减记：公司外部收购记为商誉，被收购公司的溢价。
        + 一次性收入或者支持：政府罚款，汇率收入(对冲货币的收入)，自然灾害，一次性额外收入
    + 利息
        + 不能占净利润比例太高，如果占过高，则失去偿还本金的能力
        + 原因
            + 盈利能力不强，不能提供不同质产品
            + 重资本运营：汽车，航空
            + 管理层不注重财务健康
            + 低利率存活的公司
        + 贷款不宜过高：短期贷款比例不宜过高，贷款总额不能过高
    + 税
        + 净利润增长是否因为税务减免
    + 净利润
- 其他利润
    + 毛利润：同行业成本优势
    + 营业利润：公司潜在盈利能力
    + 税前利润：利息后盈利水平
- 市盈率
    + 公司价格水平指标
    + PE = share price/earning per share
    + 戴维斯双击：pe 和 盈利同时上涨
    + 戴维斯双杀：pe 和 盈利同时下降
    + 无增长公司：10倍
    + 稳健增长公司：15倍
    + 快速增长公司：25倍
#### 资产负债表
- 利润存在形式
- 公司管理层的能力， 所得利润转为未来的资源，例如竞争力，护城河        
- 总资产 = 总负债 + 股东权益 
- 流动资产：一年之类可以转回为现金
    + 现金和短期投资：现金不能过低，公司无法支付债务， 短期投资比如投资股票，投资房产不算短期投资
    + 库存
        + 净利润应该同步升高，否则库存积压，表示公司产品竞争力下降
        + 波动大的产品说明需求稳定，库存可能贬值
    + 应收账款
        + 被迫给其他公司账期，位置处于被动
        + 可能有坏账
    + 预付费用
    + 其他
- 长期资产
    + 物业，厂房和设备
        + 不需要经常更新为好的指标
        + 更新厂房和设备用的自由资金
    + 无形资产
        + 品牌价值不能记做无形资产
        + 堆积无用商誉，不敢减记
    + 长期投资: 已买入价或者市场中低价位住，如果资产被低估，可以记为安全边际
- 流动负债
    + 应付账款：有比较长的支付账期，说明位置主动
    + 预提费用：第三方已经提供服务，可以没有付账
    + 短期债务：短期债务年利润过高，可能导致破产
    + 到期长期债务
        + 公司债务很少
        + 债务分布平均
        + 利息比较少
    + 递延所得税：到期不用支付的费用
    + 少数股权：购买公司80%的股份
- 股东权益
    + 优先股：可转化，可赎回并享受分红，已负债形式存在
    + 普通股
    + 资本公积：成立时1块钱，上市卖出10块，资本公积9块
    + 留存收益：没有发给股东，留存在公司内部
    + 库存股票：在公司回购股票产生，资本市场股本减少，股票回购优于分红   
#### 现金流量表
- 经营活动：业务运营
    + 权责发生制：产品和服务一旦提供，就记入账目
    + 折旧：有形资产的每年消耗
    + 推销：无形资产的每年摊销，比如10年使用期的专利
    + 折旧 + 摊销 是当期费用
    + 净利润
- 投资活动：长期资产，对外投资
    + 资本支出
        + 长期资产费用：多年内甩掉，先变为资产，逐年扣减当年费用
        + 比较过去10年每股总盈利和每股总资本支出，如果资本支出等于盈利，说明没有可用的发展资金
- 融资活动：发放绯红，股票回购
    + 现金股息支付
        + 高速发债，不应该发股息
        + 不能用资本投入创造额外利润，返回股息
    + 发股退股
        + 增发股票导致持股人持股贬值
        + 回购股票
    + 发债赎債
        + 发债扩张，需要归还利息，利息比较低的时候才对公司比较有利
#### 无增长型公司
- 苹果，comfortDelrgo
- PE 不要大于10
- 是否有持续分红
- 案列： 苹果, 2018年底148美金，2020年到达300美金
#### 稳健增长型公司
- 台积电，福耀玻璃
- 平均年化增长小于15%
- 未来3-5年或更长时期保持增长
- 部分伴随分红
- 不要在PE15时买入
- 研究护城河和持续增长年数
- 考虑额外分红
- 案例：台积电，2018年PE13，2020年PE19
#### 快速增长型公司
- 英伟达，脸书
- 平均年化增长大于15%
- 未来3-5年是否可以持续增长
- 大部分时间没有分红
- 不要在PE25买入
- 考虑公司的护城河和增速放缓的时间点和风险
- 案例：2018， 英伟达PE19， 2020年PE39
#### 隐藏资产型公司
- 一项或几项资产被低估
- 财务风险低，经营稳健
- 较高分红
- PE 估值不适用
#### 周期型公司
- 大宗商品，半导体，造船
- 政治影响
- 经量不投
#### 反转型公司
- 公司遭受真实打击
- 财务健康受损
- 尽量不要投

## 筛选工具
#### 股票池子
+ 大致判断是优质公司的股票池
+ 真正了解的优质公司
#### 需要股票池子
- 懂的公司，投资时获得比较高的正确概率
- 提升系统投资能力
- 同一时间，投资机会不会很多
#### 主要投资公司
- 稳健增长型
- 快速增长型
#### 公司业绩：营业额，净利润
- 营业额指标更重要，净利润提升空间有限
- 营业额提高才能提供长期增长
- 查看10年业绩，3-5年有可能是周期型公司
- 允许一定波动，整体上升
#### 判断增长率
- 历史增长数据
- 行业发展理解：公司发展环境和天花板，当前公司的市场份额
- 公司竞争优势：竞争对手能否抢走利润
- price is what you pay, value is what you get
#### 净利润率
- 产品和服务特别：不同质竞争，定价权，净利润率15%
    + 净利润率：公司好坏，大于15%
    + 毛利润率：同行公司比较
    + 苹果
        + 自称体系的IOS
        + 信息安全保护
        + APP，程序员积累
        + 产品之间的交互
        + 长期占据手机净利润的80%
- 公司效率很高
- 这两类公司的净利润更搞

#### 负载率
- 负载率小于30%，财务相对健康
- 长期负债少于4年的净利润
- 稳定的现金流，经营现金流高于或者等于年利润
- Ratio: Debit to Asset
- Balance Sheet, long term debt
- Income statement, net income
- Cashflow Statement, net incoming from continuing operation
- 特斯拉负债率42%， 高风险
#### 股本回报率
- return on equity = net income/equity
- 单位股本能产生利润的效率
- 股本回报率大于15%
- AAPL
    + 利润: 55.3
    + 股本: 90.5
    + 股本回报率：55.92
- 股票回购，用债务降低股本
- 考虑因素
    + 运营资金充足， current asset 大于 current liability
        + Balance sheet, Total Current Asset, Total Current Liability
    + 长期负债是否4年内还清，总负债率在30%一下

## 战略
#### 战略理解
- 提供什么价值
- 如何提供价值
- 如何收获利润
- 使用何种方式提供价值并收获利润
- 公司战略为公司为实现目标使用的战略
    - 小米: 公司目标, 流量的增值服务，利润来自增值服务，策略是高性价比
    - 樊登读书: 书中智慧赋予生活，利润来源会员年费， 策略是让加盟商成功扩大公司    
- 范围：总体， 竞争，营销，品牌，融资，人才，技术，资源
- 公司战略是更具情况，选择经营和发展的产品和服务，构建竞争力并实现公司目标
#### 三个战略研究
- 总体战略
- 竞争战略：竞争优势，如何增强优势并获胜
- 资源战略：积累什么资源，如何积累
- 樊登读书
    + 总体战略：通过赋能加盟山，获得公司成功
    + 竞争战略：经典智慧使用，能把晦涩的智慧显得浅显易懂
    + 资源战略：加盟商，讲知识的能力和状态
#### 理解商业模式
- 理解公司战略的前提
- 公司战略的决策依据
- 资源调配依据
#### 理解公司战略
- 判断公司是否前进
- 判断公司护城河依据
- 判断公司长期价值的依据：利润和价值是否匹配

|维度|  樊登读书 | 逻辑思维 |
|-----|--------| --------|
| 提供价值 | 实用价值 | 冷门价值 |
|收获利润 | 会员续订 | 项目利润 |
|护城河 |浅显易懂，有趣 | 专业性的普通话教育 |
|公司价值| 生活场景使用 | 专业领域使用 |

#### 过去和未来战略
- 过去战略：积累资源和优势
- 未来战略：迭代方向和方式

#### 例外情况
- 公司管理层没有想明白商业模式
    + 逻辑思维做得到APP之前
    + 互联网公司 
- 公司管理层没有明白战略
    + 谷歌竞价广告
- 公司管理层知道商业模式和战略，但是做不到
    + 海底捞模仿者


## 公司团队
#### 结构
- 董事会：重要决策，人员任免
    + 董事会主席
    + Executive Director： 参与经营管理
    + Non-Executive Director: 不参与经营管理
    + Independent Directory: 不参加经营管理，跟公司没有任何利益关系
- 执行官：经营管理

## 行业上下游

- 在议价方面，谁更有优势, 
- 有没有跟供应商达成长期的互利合作关系，能否在控制成本的前提下保证价格相对稳定
#### 下游客户
- 谁是公司的主要客户
- 在议价方面，谁更有优势
- 客户构成是否太过集中，头部几家客户占去了总营业额的多少百分比

- 资源的布局有没有让公司失去聚焦

## 竞争分析
#### 竞争格局分析
- 主要竞争者
    + 垄断：一个最大公司占据主要市场
        + 特点：一个最大公司拥有技术壁垒，规模效应，形成垄断，占据主要行业利润，并有用定价权
        + 竞争门槛较高
        + 利润率较高
        + 例子： MSFT， ASML
    + 寡头竞争：几个势均力敌的公司形成第一梯队，占据主要市场
        + 第一梯队公司各有自己的优势, 有一定竞争优势
        + 竞争门槛比较高
        + 利润率可能比较高也可能比较低
        + 例子: 华为, 小米, OPPO
    + 完全竞争：非常多的公司，集体占据主要市场
        + 差异化竞争
            + 产品和服务相对差异化，难以大规模合并，公司与一定定价权
            + 竞争门槛较低
            + 利润率相对高
            + 例子：奢侈品行业，健康食品
        + 同质化竞争
            + 产品和服务同质化，公司没有定价权
            + 竞争门槛低
            + 利润率低
            + 例子：手机膜，手机壳
- 竞争激烈程度分析
    - 大体量行业的激烈竞争
        + 巨额体量资本注入
        + 拥有雄心和能力的强有力竞争者
        + 不断发展下的新竞争者
        + 例子：汽车行业
    - 小体量行业竞争缺乏
        + 体量太小容不下大资本
        + 回报太小能人不看重
        + 缺乏强有力的竞争者
        + 例子：水电维修
- 公司利润状况
    + 高速增长期：行业体量迅速变大，所有公司盈利都相对轻松
        + 竞争小， 行业最强者面临更替，竞争力不佳的公司未来有破产风险
        + 例子：智能家电，新能源汽车, 电商的拼多多
    + 缓慢增长期：行业体量稳定但增长缓慢，市场份额争夺成为主要目标
        + 主要争抢市场份额
        + 例子：供水供电
    + 稳定，收缩期： 行业进入萎缩，被新行业提点，整体利润下降
        + 竞争小，产能过剩，公司有可能破产
#### 竞争对手分析
- 查找对手
    + gurufocus： analysis， competitive comparison
    + google：major competitors
- 相对竞争优势的分析
    + 商业模式
        + 提供的价值的未来总体量更大
            + 樊登vs得到，谷歌vs百度
        + 哪家公司的收获利润方式更好
            + costco vs walmart
        + 哪家公司战略更好
            + 未来方向
            + 公司靠什么取胜
            + 积累竞争优势
            + facebook vs twitter
    + 产品和服务
        + 有没有很好解决问题
            + zoom vs cisco webex
        + 用户体验
            + 海底捞 vs 其他火锅
        + 用户忠诚度
            + 苹果 vs 小米
    + 销售体系
        + 渠道，流量和客户群的掌控能力
            + 腾讯 vs 其他互联网公司
        + 销售团队
            + 阿里 vs 腾讯
    + 核心团队
        + 团队能力：是否实现目标
        + 团队协作
    + 成本控制
        + 成本是否有优势
            + 小米生态 vs IOT创业公司
        + 是否有进一步消减成本的能力的聚焦
    + 资金
        + 资金积累
        + 资金调用能力
        + facebook vs Snapchat
    +行业上下游
        + 上下游是否产生更强竞争力
        + 聚焦
- 同一行业最重要环节的演化
    + 产品啊，流量，资金，数据
#### 竞争终局
- 含义
   + 在一轮竞争过后，相对稳定的竞争格局
   + 终局不是永久的，是一段时间内的相对平衡
   + 终局可能被完全打破，进入下一个竞争
   + 滴滴和快滴合并， 波音vs麦道，波音vs空客
- 意义
    + 帮助我们了解竞争可能走向
    + 帮助我们控制承担风险        
    + 帮助我们做投资决策
- 常见竞争终局    
    + 垄断公司打压新的竞争者， Grab vs Uber, Grab vs Gojek
    + 寡头竞争下的快速增长公司
    + 完全竞争下的快速增长公司
        + 同质化竞争利润低，非标准化
- 判断方法
    + 类似案例历史
    + 主要分析竞争者的护城河
    + 数据趋势

## 风险
#### 风险的含义
- 资本安全角度：损失本金的可能
- 从投资决策的角度：影响投资决定发送误判，或者公司价值的因素的总和

#### 风险的误解
- 波动不是风险
- 除非在没有计划好投资时间跨度的前提下

#### 投资 vs 投机 vs 赌博
- 投资： 大概率获利，本金比较安全
- 投机： 不是大概率获利或者本金不安全
- 赌博：绩效概率获利，本金极度不安全

#### 投资的两个维度
- 获利概率
- 本金安全 (乘法题 vs 加法题)
- 做法
    + 深入研究，形成系统
    + 获得概率优势和本金安全
    + 从投机和赌博转为投资

#### 拥有投资系统
- 可重复：效率提升
- 可提升: 持续降低风险，提高回报
- 自恰：可重复提升的基石，拥有底层逻辑，可硬度大部分情况和对新情况判断
- 做法
    + 建立自己的投资系统
    + 先模仿，再调整

#### 主要风险类别
- 投资者
    + 认知风险：对公司相关的重要方面发生误判， e.g. 竞争优势，盈利能力，财务能力
        + 完全认知不存在，要在有限时间内排出大部分认知风险
        + 排出大部分风险后再能力圈内投资
        + 盈利能力的误判：互联网公司
        + 竞争优势的误判：AMD两个领域第二名
        + 经纪环境：巴菲特加仓航空股
    + 心态风险：因为心理因素导致不能执行自己的端盘
        + 恐惧深跌，贪图非理性生长，厌恶亏损，比较心态，神话个人和团队
        + 应对：锻炼心态，借助peer group的力量
    + 价值投资知易行难
        + 延迟满足：需要耐心等待合适的机会
        + 不从众：言语做出大部分人不同的决定
- 竞争风险
    + 公司相对竞争优势发生变化
    + 不断跟踪公司竞争对手变化
    + 例如：诺基亚垄断，百度公司文化变化，costco挑战walmart
    + 对竞争的正确分析是长期投资的基石，也是最困难的部分
- 行业风险
    + 行业的体量和利润变化
    + 行业整体被替代，比如胶卷，传统手机，中国手机市场进入收缩期，石油行业
- 政策风险
    + 政策原因导致公司盈利能力
    + 行业优惠政策撤销，例如：中国光伏行业，中国停止发放游戏版号
    + 关税的增加，2018 中美贸易战下的外贸公司
    + 例如: 2008经纪危机, 希腊, 俄罗斯石油行业
+ 控制好风险，顺势额为，在趋势面前，最优秀的团队也可能无计可施    
- 财务风险
	+ 流动性风险
		+ current ratio = current asset/current liability
		+ quick ratio = (cash  + markable securities + receivables) / current liabilities 
		+ cash ratio = (cash  + markable securities) / current liabilities 
		+ ratio > 1, 三个指标逐渐严格，看公司是否有短期还债的能力
	+ 总债务风险
	    + 债务水平
	        - 四个个指标越小越好
            - debit to asset = total debt/total asset
            - debt to equity = total debt/total equity
            - debt to capital = total debt /(total debt + total shareholder's equity)
            - Financial leverage = average total asset / average total equity, average指一年之内
        + 偿还能力
            - interest coverage = earning before interest and taxtes / interest payment, 数值越大越好
            - fixed charge coverage =  earning before interest and taxtes + lease payment / interest payment, 数值越大越好
            - debt payment = total debt / net earnings, 小于4
        + 依靠债务提高收益率
        + 依靠债务维持竞争力
        + 依靠债务续命
	+ 利润率风险
	    + 比较同行业公司和竞争对手情况
	    + 比较和公司过去
	    + cross profit margin = gross profit / revenue, 成本控制，比对手高
	    + operating margin = earning before interest and tax / revenue， 运营能力，比对手高
	    + pretax margin = earnings before tax / revenue， 比对手高
	    + net profit margin = net earning / revenue, 比对手高
	    + return on asset = net earnings / average total asset, 比对手高
	    + return on equity = net earnings / average total equity, 大于15%
	+ 运营风险
	    + receivables turnover = annual sales / average receivable, 一年卖多少轮货，比对手高
	    + days of sales outstanding = 365 / receivables turnover, 比对手低
	    + inventory turnover = cost of goods sold / average inventory, 一年买多少轮货，比对手高
	    + days of inventory on hand = 365 / inventory turnover, 比对手低
	    + payable turnover = purchases / average trade payables, 比对手低，欠供货商的钱
	    + number of days payable = 365 / payable turnover, 比对手高，付款期限，越高越好
	    + cash convention cycle = days of sales outstanding + days of inventory on hand - number of days payable, 比对手低，比自己过去更低
- 数量分析 vs 质量分析
    + 数量分析1家公司是否过关，值不值得花更多时间研究
    + 数量分析是质量分析的数据化，帮助理解和分析公司，不要过于依赖和过度解读
    + NCAV = cash & cash equivalent + partial receivable + partial inventory - total liability 来计算安全边际
- 管理层风险
    + 最高决策人换人， 例如台积电张忠谋，王石，乔布斯
    + 团队分裂， 内讧，党争
    + 资本控制
- 市场风险
    + 利息大幅提高：反相关
    + 流动性不够：流动盘总量，平均每天交易量大小
    + 其他资产吸收资金
- 顾客风险
    + 需求变化， 电脑到手机，
    + 行为认知变化
    + 行为方式
