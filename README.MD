# 股票投资入门

## 能力圈
- 巴菲特，价值投资，760亿， 20%年投资回报率
- 判断准确度高，区别投资和投机，避免重大损失
- 错误做法
    + 投资热门股票
    + 不做产盘和服务研究
    + 购买别人推荐的股票
    + 跟风购买别人挣钱的股票
- 对产品服务熟悉，对公司有调查，对自己的投资有信心(70%)
***

## 安全边际
- 安全边际是价值和价格的容错空间，不要加杠杆！
- 减少损失，扩大受益
- 确定性和安全边际成比例
    + 确定性越低，需要留出更大的安全边际
- 出现时机：市场，行业，公司危机
- 常见错误：没有对比价值和价格, 斗鱼IPO投资失败案例
#### 明确投资目标
- 长期投资目标
- 个股投资目标, 年回报率
- 目标设定
    + 快，小，市场波动
    + 长，大，公司价值提升或者低估
#### 参与者
- 华尔街
    + 倾向看涨
- 机构投资者
    + 考核机制，相对回报（相对同行的盈利状况）
    + 投资限制
- 监管部门
    + 关注下跌
#### 价格的决定因素
+ 公司的盈利能力：长期平均价格
+ 供求关系：短期价格
#### 催化剂
- 公司盈利回归
- 盈利增长
- 新业务的商业模式
- 短期催化剂：比如公司年度产品
- 长期催化剂：模式或者盈利能力，比如云计算
#### 能力圈 vs 安全边际
- 能力圈：投资对的公司
- 安全边际：投资对的价格
***

## 护城河
- 保护自己未来盈利能力的积累，相对竞争优势的累积
    + 针对现有竞争者
    + 针对潜在竞争者
- 类型
    + 专利型：芯片，制药
    + 品牌：可口可乐，穆迪
    + 技术：台积电，英伟达
    + 经营效率型：costco，小米
    + 网路效应：脸书
- 深度，相对竞争优势
    + 公司本身的积累
    + 竞争对手的积累
- 商业模式
    + 提供价值
    + 获利方式
- 概率
    + 增加判断正确的概率
    + 减低判断错误的概率
***


## 应用
- 价值投资的思维方式
    + 投资时候比较少，多次练习
    + 积累正向反馈，加强定力
- 安全边际
    + 给自己留出容错空间或者预留第二套方案
- 能力圈
    + 加强判断力
    + 提高判断正确性
    + 判断力低的时候，留出更大的安全控件或者不行动
- 护城河
    + 持续累积自己的优势
    + 比如，阅读，健身，学习

## 财务报表
- 损益表：分析公司盈利能力
- 资产负债表：分析同事财务状况，财务风险
- 现金流量表：公司流动性能力
#### 报表查询
- sec.gov
#### 财务报表
- 倾向于持久和稳定的利润
- 避免周期型公司，不确定因素，不推荐非行业专业人士
- 财务数据： gurufocus.com， 查看10年以上的财务数据
#### 优点和缺点
- 标准化，信息透明
- 数量分析为主，质量取决于经营模式，管理，行业，风险
#### 损益表
- 市场定价来源
- 衡量公司价值
    + e.g. 糖果店
    + 营业额： 提价能力，销量，产品线
    + 产品成本, 毛利率: 产品成本，原材料和制作工艺
        + 控制成本，供应链地位，原材料生成
        + 价值溢价，专利和技术
        + 品牌和认可
        + 对手退出，形成寡头：比如数字货币交易所
    + 运营成本: 房租，人工，广告，折旧
        + 销售和一般费用
            + 营销成本升高说明竞争力下降，营销指向渠道和品牌
            + 工资过高说明结构问题
        + 研发费用
            + 非科技公司： 预期比较低
            + 科技公司：足够投入，长期积累
        + 折旧
            + 不能占营业额太多
    + 特别项目
        + 出售资产：聚焦优势项目或者回收现金流，反向推论买入的资产是否恰当 
        + 商誉减记：公司外部收购记为商誉，被收购公司的溢价。
        + 一次性收入或者支持：政府罚款，汇率收入(对冲货币的收入)，自然灾害，一次性额外收入
    + 利息
        + 不能占净利润比例太高，如果占过高，则失去偿还本金的能力
        + 原因
            + 盈利能力不强，不能提供不同质产品
            + 重资本运营：汽车，航空
            + 管理层不注重财务健康
            + 低利率存活的公司
        + 贷款不宜过高：短期贷款比例不宜过高，贷款总额不能过高
    + 税
        + 净利润增长是否因为税务减免
    + 净利润
- 其他利润
    + 毛利润：同行业成本优势
    + 营业利润：公司潜在盈利能力
    + 税前利润：利息后盈利水平
- 市盈率
    + 公司价格水平指标
    + PE = share price/earning per share
    + 戴维斯双击：pe 和 盈利同时上涨
    + 戴维斯双杀：pe 和 盈利同时下降
    + 无增长公司：10倍
    + 稳健增长公司：15倍
    + 快速增长公司：25倍
#### 资产负债表
- 利润存在形式
- 公司管理层的能力， 所得利润转为未来的资源，例如竞争力，护城河        
- 总资产 = 总负债 + 股东权益 
- 流动资产：一年之类可以转回为现金
    + 现金和短期投资：现金不能过低，公司无法支付债务， 短期投资比如投资股票，投资房产不算短期投资
    + 库存
        + 净利润应该同步升高，否则库存积压，表示公司产品竞争力下降
        + 波动大的产品说明需求稳定，库存可能贬值
    + 应收账款
        + 被迫给其他公司账期，位置处于被动
        + 可能有坏账
    + 预付费用
    + 其他
- 长期资产
    + 物业，厂房和设备
        + 不需要经常更新为好的指标
        + 更新厂房和设备用的自由资金
    + 无形资产
        + 品牌价值不能记做无形资产
        + 堆积无用商誉，不敢减记
    + 长期投资: 已买入价或者市场中低价位住，如果资产被低估，可以记为安全边际
- 流动负债
    + 应付账款：有比较长的支付账期，说明位置主动
    + 预提费用：第三方已经提供服务，可以没有付账
    + 短期债务：短期债务年利润过高，可能导致破产
    + 到期长期债务
        + 公司债务很少
        + 债务分布平均
        + 利息比较少
    + 递延所得税：到期不用支付的费用
    + 少数股权：购买公司80%的股份
- 股东权益
    + 优先股：可转化，可赎回并享受分红，已负债形式存在
    + 普通股
    + 资本公积：成立时1块钱，上市卖出10块，资本公积9块
    + 留存收益：没有发给股东，留存在公司内部
    + 库存股票：在公司回购股票产生，资本市场股本减少，股票回购优于分红   
#### 现金流量表
- 经营活动：业务运营
    + 权责发生制：产品和服务一旦提供，就记入账目
    + 折旧：有形资产的每年消耗
    + 推销：无形资产的每年摊销，比如10年使用期的专利
    + 折旧 + 摊销 是当期费用
    + 净利润
- 投资活动：长期资产，对外投资
    + 资本支出
        + 长期资产费用：多年内甩掉，先变为资产，逐年扣减当年费用
        + 比较过去10年每股总盈利和每股总资本支出，如果资本支出等于盈利，说明没有可用的发展资金
- 融资活动：发放绯红，股票回购
    + 现金股息支付
        + 高速发债，不应该发股息
        + 不能用资本投入创造额外利润，返回股息
    + 发股退股
        + 增发股票导致持股人持股贬值
        + 回购股票
    + 发债赎債
        + 发债扩张，需要归还利息，利息比较低的时候才对公司比较有利
