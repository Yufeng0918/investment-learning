# 股票投资入门

## 能力圈
- 巴菲特，价值投资，760亿， 20%年投资回报率
- 判断准确度高，区别投资和投机，避免重大损失
- 错误做法
    + 投资热门股票
    + 不做产盘和服务研究
    + 购买别人推荐的股票
    + 跟风购买别人挣钱的股票
- 对产品服务熟悉，对公司有调查，对自己的投资有信心(70%)
***

## 安全边际
- 安全边际是价值和价格的容错空间，不要加杠杆！
- 减少损失，扩大受益
- 确定性和安全边际成比例
    + 确定性越低，需要留出更大的安全边际
- 出现时机：市场，行业，公司危机
- 常见错误：没有对比价值和价格, 斗鱼IPO投资失败案例
#### 明确投资目标
- 长期投资目标
- 个股投资目标, 年回报率
- 目标设定
    + 快，小，市场波动
    + 长，大，公司价值提升或者低估
#### 参与者
- 华尔街
    + 倾向看涨
- 机构投资者
    + 考核机制，相对回报（相对同行的盈利状况）
    + 投资限制
- 监管部门
    + 关注下跌
#### 价格的决定因素
+ 公司的盈利能力：长期平均价格
+ 供求关系：短期价格
#### 催化剂
- 公司盈利回归
- 盈利增长
- 新业务的商业模式
- 短期催化剂：比如公司年度产品
- 长期催化剂：模式或者盈利能力，比如云计算
#### 能力圈 vs 安全边际
- 能力圈：投资对的公司
- 安全边际：投资对的价格
***

## 护城河
- 保护自己未来盈利能力的积累，相对竞争优势的累积
    + 针对现有竞争者
    + 针对潜在竞争者
- 类型
    + 专利型：芯片，制药
    + 品牌：可口可乐，穆迪
    + 技术：台积电，英伟达
    + 经营效率型：costco，小米
    + 网路效应：脸书
- 深度，相对竞争优势
    + 公司本身的积累
    + 竞争对手的积累
- 商业模式
    + 提供价值
    + 获利方式
- 概率
    + 增加判断正确的概率
    + 减低判断错误的概率
***


## 应用
- 价值投资的思维方式
    + 投资时候比较少，多次练习
    + 积累正向反馈，加强定力
- 安全边际
    + 给自己留出容错空间或者预留第二套方案
- 能力圈
    + 加强判断力
    + 提高判断正确性
    + 判断力低的时候，留出更大的安全控件或者不行动
- 护城河
    + 持续累积自己的优势
    + 比如，阅读，健身，学习

## 财务报表
- 损益表：分析公司盈利能力
- 资产负债表：分析同事财务状况，财务风险
- 现金流量表：公司流动性能力
#### 报表查询
- sec.gov
#### 财务报表
- 倾向于持久和稳定的利润
- 避免周期型公司，不确定因素，不推荐非行业专业人士
- 财务数据： gurufocus.com， 查看10年以上的财务数据
#### 优点和缺点
- 标准化，信息透明
- 数量分析为主，质量取决于经营模式，管理，行业，风险
#### 损益表
- 衡量公司价值
    + e.g. 糖果店
    + 营业额： 提价能力，销量，产品线
    + 销售成本: 产品成本，原材料和制作工艺
        + 毛利率
            + 控制成本，供应链地位，原材料生成
            + 价值溢价，专利和技术
            + 品牌和认可
            + 对手退出，形成寡头：比如数字货币交易所
    + 运营成本: 房租，人工，广告，折旧
        + 销售和一般费用
            + 营销成本升高说明竞争力下降，营销指向渠道和品牌
            + 工资过高说明结构问题
        + 研发费用
            + 非科技公司： 预期比较低
            + 科技公司：足够投入，长期积累
        + 折旧
            + 不能占营业额太多
    + 特别项目
    + 税
    + 净利润
- 市场定价来源
