# 财务自由

## 1. 纲领
#### 原则

**拥有“生钱资产”**，让“生钱资产”生的钱大于你的生活总支出 生钱资产

**减少“耗钱资产”**



#### 工具

三大核心工具： **企业，股票，房地产**

三个辅助工具分别为：**分级基金 A，逆回购，货币基金**



#### 准则

+ 正向现金流
+ 制定投资标准
+ 全球化配置

+ 长期投资



#### 禁令

+ 不违法，不投机， 投入资金搏取随机性收益的行为。
+ 不预测趋势
+ 不执著于投资工具

## 安全边际
- 安全边际是价值和价格的容错空间，不要加杠杆！
- 减少损失，扩大受益
- 确定性和安全边际成比例
    + 确定性越低，需要留出更大的安全边际
- 出现时机：市场，行业，公司危机
- 常见错误：没有对比价值和价格, 斗鱼IPO投资失败案例
#### 明确投资目标
- 长期投资目标
- 个股投资目标, 年回报率
- 目标设定
    + 快，小，市场波动
    + 长，大，公司价值提升或者低估
#### 参与者
- 华尔街
    + 倾向看涨
- 机构投资者
    + 考核机制，相对回报（相对同行的盈利状况）
    + 投资限制
- 监管部门
    + 关注下跌
#### 价格的决定因素
+ 公司的盈利能力：长期平均价格
+ 供求关系：短期价格
#### 催化剂
- 公司盈利回归
- 盈利增长
- 新业务的商业模式
- 短期催化剂：比如公司年度产品
- 长期催化剂：模式或者盈利能力，比如云计算
#### 能力圈 vs 安全边际
- 能力圈：投资对的公司
- 安全边际：投资对的价格

## 财务报表
- 损益表：分析公司盈利能力
- 资产负债表：分析同事财务状况，财务风险
- 现金流量表：公司流动性能力
#### 报表查询
- sec.gov
#### 财务报表
- 倾向于持久和稳定的利润
- 避免周期型公司，不确定因素，不推荐非行业专业人士
- 财务数据： gurufocus.com， 查看10年以上的财务数据
#### 优点和缺点
- 标准化，信息透明
- 数量分析为主，质量取决于经营模式，管理，行业，风险
#### 损益表
市场定价来源, 衡量公司价值

+ e.g. 糖果店
+ 营业额： 提价能力，销量，产品线
+ 产品成本, 毛利率: 产品成本，原材料和制作工艺
    + 控制成本，供应链地位，原材料生成
    + 价值溢价，专利和技术
    + 品牌和认可
    + 对手退出，形成寡头：比如数字货币交易所
+ 运营成本: 房租，人工，广告，折旧
    + 销售和一般费用
        + 营销成本升高说明竞争力下降，营销指向渠道和品牌
        + 工资过高说明结构问题
    + 研发费用
        + 非科技公司： 预期比较低
        + 科技公司：足够投入，长期积累
    + 折旧
        + 不能占营业额太多
+ 特别项目
    + 出售资产：聚焦优势项目或者回收现金流，反向推论买入的资产是否恰当 
    + 商誉减记：公司外部收购记为商誉，被收购公司的溢价。
    + 一次性收入或者支持：政府罚款，汇率收入(对冲货币的收入)，自然灾害，一次性额外收入
+ 利息
    + 不能占净利润比例太高，如果占过高，则失去偿还本金的能力
    + 原因
        + 盈利能力不强，不能提供不同质产品
        + 重资本运营：汽车，航空
        + 管理层不注重财务健康
        + 低利率存活的公司
    + 贷款不宜过高：短期贷款比例不宜过高，贷款总额不能过高
+ 税
    + 净利润增长是否因为税务减免
+ 净利润

其他利润
+ 毛利润：同行业成本优势
+ 营业利润：公司潜在盈利能力
+ 税前利润：利息后盈利水平

市盈率
+ 公司价格水平指标
+ PE = share price/earning per share
+ 戴维斯双击：pe 和 盈利同时上涨
+ 戴维斯双杀：pe 和 盈利同时下降
+ 无增长公司：10倍
+ 稳健增长公司：15倍
+ 快速增长公司：25倍

#### 资产负债表
利润存在形式, 公司管理层的能力， 所得利润转为未来的资源，例如竞争力，护城河        

总资产 = 总负债 + 股东权益 

流动资产：一年之类可以转回为现金
+ 现金和短期投资：现金不能过低，公司无法支付债务， 短期投资比如投资股票，投资房产不算短期投资
+ 库存
    + 净利润应该同步升高，否则库存积压，表示公司产品竞争力下降
    + 波动大的产品说明需求稳定，库存可能贬值
+ 应收账款
    + 被迫给其他公司账期，位置处于被动
    + 可能有坏账
+ 预付费用
+ 其他

长期资产
+ 物业，厂房和设备
    + 不需要经常更新为好的指标
    + 更新厂房和设备用的自由资金
+ 无形资产
    + 品牌价值不能记做无形资产
    + 堆积无用商誉，不敢减记
+ 长期投资: 已买入价或者市场中低价位住，如果资产被低估，可以记为安全边际

流动负债
+ 应付账款：有比较长的支付账期，说明位置主动
+ 预提费用：第三方已经提供服务，可以没有付账
+ 短期债务：短期债务年利润过高，可能导致破产
+ 到期长期债务
    + 公司债务很少
    + 债务分布平均
    + 利息比较少
+ 递延所得税：到期不用支付的费用
+ 少数股权：购买公司80%的股份

股东权益
+ 优先股：可转化，可赎回并享受分红，已负债形式存在
+ 普通股
+ 资本公积：成立时1块钱，上市卖出10块，资本公积9块
+ 留存收益：没有发给股东，留存在公司内部
+ 库存股票：在公司回购股票产生，资本市场股本减少，股票回购优于分红   

#### 现金流量表
经营活动：业务运营
+ 权责发生制：产品和服务一旦提供，就记入账目
+ 折旧：有形资产的每年消耗
+ 推销：无形资产的每年摊销，比如10年使用期的专利
+ 折旧 + 摊销 是当期费用
+ 净利润

投资活动：长期资产，对外投资
+ 资本支出
    + 长期资产费用：多年内甩掉，先变为资产，逐年扣减当年费用
    + 比较过去10年每股总盈利和每股总资本支出，如果资本支出等于盈利，说明没有可用的发展资金

融资活动：发放绯红，股票回购
+ 现金股息支付
    + 高速发债，不应该发股息
    + 不能用资本投入创造额外利润，返回股息
+ 发股退股
    + 增发股票导致持股人持股贬值
    + 回购股票
+ 发债赎債
    + 发债扩张，需要归还利息，利息比较低的时候才对公司比较有利

#### 无增长型公司
- 苹果，comfortDelrgo
- PE 不要大于10
- 是否有持续分红
- 案列： 苹果, 2018年底148美金，2020年到达300美金
#### 稳健增长型公司
- 台积电，福耀玻璃
- 平均年化增长小于15%
- 未来3-5年或更长时期保持增长
- 部分伴随分红
- 不要在PE15时买入
- 研究护城河和持续增长年数
- 考虑额外分红
- 案例：台积电，2018年PE13，2020年PE19
#### 快速增长型公司
- 英伟达，脸书
- 平均年化增长大于15%
- 未来3-5年是否可以持续增长
- 大部分时间没有分红
- 不要在PE25买入
- 考虑公司的护城河和增速放缓的时间点和风险
- 案例：2018， 英伟达PE19， 2020年PE39
#### 隐藏资产型公司
- 一项或几项资产被低估
- 财务风险低，经营稳健
- 较高分红
- PE 估值不适用
#### 周期型公司
- 大宗商品，半导体，造船
- 政治影响
- 经量不投
#### 反转型公司
- 公司遭受真实打击
- 财务健康受损
- 尽量不要投

## 战略

#### 战略理解
- 提供什么价值
- 如何提供价值
- 如何收获利润
- 使用何种方式提供价值并收获利润
- 公司战略为公司为实现目标使用的战略
    - 小米: 公司目标, 流量的增值服务，利润来自增值服务，策略是高性价比
    - 樊登读书: 书中智慧赋予生活，利润来源会员年费， 策略是让加盟商成功扩大公司    
- 范围：总体， 竞争，营销，品牌，融资，人才，技术，资源
- 公司战略是更具情况，选择经营和发展的产品和服务，构建竞争力并实现公司目标
#### 三个战略研究
- 总体战略
- 竞争战略：竞争优势，如何增强优势并获胜
- 资源战略：积累什么资源，如何积累
- 樊登读书
    + 总体战略：通过赋能加盟山，获得公司成功
    + 竞争战略：经典智慧使用，能把晦涩的智慧显得浅显易懂
    + 资源战略：加盟商，讲知识的能力和状态
#### 理解商业模式
- 理解公司战略的前提
- 公司战略的决策依据
- 资源调配依据
#### 理解公司战略
- 判断公司是否前进
- 判断公司护城河依据
- 判断公司长期价值的依据：利润和价值是否匹配

|维度|  樊登读书 | 逻辑思维 |
|-----|--------| --------|
| 提供价值 | 实用价值 | 冷门价值 |
|收获利润 | 会员续订 | 项目利润 |
|护城河 |浅显易懂，有趣 | 专业性的普通话教育 |
|公司价值| 生活场景使用 | 专业领域使用 |

#### 过去和未来战略
- 过去战略：积累资源和优势
- 未来战略：迭代方向和方式

#### 例外情况

    + 逻辑思维做得到APP之前

- 公司管理层没有明白战略
    + 谷歌竞价广告

    + 海底捞模仿者



## 公司团队

#### 结构
- 董事会：重要决策，人员任免
    + 董事会主席
    + Executive Director： 参与经营管理
    + Non-Executive Director: 不参与经营管理
    + Independent Directory: 不参加经营管理，跟公司没有任何利益关系
- 执行官：经营管理

## 竞争分析
#### 竞争格局分析
- 主要竞争者
    + 垄断：一个最大公司占据主要市场
        + 特点：一个最大公司拥有技术壁垒，规模效应，形成垄断，占据主要行业利润，并有用定价权
        + 竞争门槛较高
        + 利润率较高
        + 例子： MSFT， ASML
    + 寡头竞争：几个势均力敌的公司形成第一梯队，占据主要市场
        + 第一梯队公司各有自己的优势, 有一定竞争优势
        + 竞争门槛比较高
        + 利润率可能比较高也可能比较低
        + 例子: 华为, 小米, OPPO
    + 完全竞争：非常多的公司，集体占据主要市场
        + 差异化竞争
            + 产品和服务相对差异化，难以大规模合并，公司与一定定价权
            + 竞争门槛较低
            + 利润率相对高
            + 例子：奢侈品行业，健康食品
        + 同质化竞争
            + 产品和服务同质化，公司没有定价权
            + 竞争门槛低
            + 利润率低
            + 例子：手机膜，手机壳
- 竞争激烈程度分析
    - 大体量行业的激烈竞争
        + 巨额体量资本注入
        + 拥有雄心和能力的强有力竞争者
        + 不断发展下的新竞争者
        + 例子：汽车行业
    - 小体量行业竞争缺乏
        + 体量太小容不下大资本
        + 回报太小能人不看重
        + 缺乏强有力的竞争者
        + 例子：水电维修
- 公司利润状况
    + 高速增长期：行业体量迅速变大，所有公司盈利都相对轻松
        + 竞争小， 行业最强者面临更替，竞争力不佳的公司未来有破产风险
        + 例子：智能家电，新能源汽车, 电商的拼多多
    + 缓慢增长期：行业体量稳定但增长缓慢，市场份额争夺成为主要目标
        + 主要争抢市场份额
        + 例子：供水供电
    + 稳定，收缩期： 行业进入萎缩，被新行业提点，整体利润下降
        + 竞争小，产能过剩，公司有可能破产
#### 竞争对手分析
- 查找对手
    + gurufocus： analysis， competitive comparison
    + google：major competitors
- 相对竞争优势的分析
    + 商业模式
        + 提供的价值的未来总体量更大
            + 樊登vs得到，谷歌vs百度
        + 哪家公司的收获利润方式更好
            + costco vs walmart
        + 哪家公司战略更好
            + 未来方向
            + 公司靠什么取胜
            + 积累竞争优势
            + facebook vs twitter
    + 产品和服务
        + 有没有很好解决问题
            + zoom vs cisco webex
        + 用户体验
            + 海底捞 vs 其他火锅
        + 用户忠诚度
            + 苹果 vs 小米
    + 销售体系
        + 渠道，流量和客户群的掌控能力
            + 腾讯 vs 其他互联网公司
        + 销售团队
            + 阿里 vs 腾讯
    + 核心团队
        + 团队能力：是否实现目标
        + 团队协作
    + 成本控制
        + 成本是否有优势
            + 小米生态 vs IOT创业公司
        + 是否有进一步消减成本的能力的聚焦
    + 资金
        + 资金积累
        + 资金调用能力
        + facebook vs Snapchat
    +行业上下游
        + 上下游是否产生更强竞争力
        + 聚焦
- 同一行业最重要环节的演化
    + 产品啊，流量，资金，数据
#### 竞争终局
- 含义
   + 在一轮竞争过后，相对稳定的竞争格局
   + 终局不是永久的，是一段时间内的相对平衡
   + 终局可能被完全打破，进入下一个竞争
   + 滴滴和快滴合并， 波音vs麦道，波音vs空客
- 意义

    + 完全竞争下的快速增长公司

    + 数据趋势

## 风险
#### 风险的含义
- 资本安全角度：损失本金的可能
- 从投资决策的角度：影响投资决定发送误判，或者公司价值的因素的总和

#### 风险的误解
- 波动不是风险
- 除非在没有计划好投资时间跨度的前提下

#### 投资 vs 投机 vs 赌博
- 投资： 大概率获利，本金比较安全
- 投机： 不是大概率获利或者本金不安全
- 赌博：绩效概率获利，本金极度不安全

#### 投资的两个维度
- 获利概率
- 本金安全 (乘法题 vs 加法题)
- 做法
    + 深入研究，形成系统
    + 获得概率优势和本金安全
    + 从投机和赌博转为投资

#### 拥有投资系统
- 可重复：效率提升
- 可提升: 持续降低风险，提高回报
- 自恰：可重复提升的基石，拥有底层逻辑，可硬度大部分情况和对新情况判断
- 做法
    + 建立自己的投资系统
    + 先模仿，再调整

#### 主要风险类别
- 投资者
    + 认知风险：对公司相关的重要方面发生误判， e.g. 竞争优势，盈利能力，财务能力
        + 完全认知不存在，要在有限时间内排出大部分认知风险
        + 排出大部分风险后再能力圈内投资
        + 盈利能力的误判：互联网公司
        + 竞争优势的误判：AMD两个领域第二名
        + 经纪环境：巴菲特加仓航空股
    + 心态风险：因为心理因素导致不能执行自己的端盘
        + 恐惧深跌，贪图非理性生长，厌恶亏损，比较心态，神话个人和团队
        + 应对：锻炼心态，借助peer group的力量
    + 价值投资知易行难
        + 延迟满足：需要耐心等待合适的机会
        + 不从众：言语做出大部分人不同的决定
- 竞争风险
    + 公司相对竞争优势发生变化
    + 不断跟踪公司竞争对手变化
    + 例如：诺基亚垄断，百度公司文化变化，costco挑战walmart
    + 对竞争的正确分析是长期投资的基石，也是最困难的部分
- 行业风险
    + 行业的体量和利润变化
    + 行业整体被替代，比如胶卷，传统手机，中国手机市场进入收缩期，石油行业
- 政策风险
    + 政策原因导致公司盈利能力
    + 行业优惠政策撤销，例如：中国光伏行业，中国停止发放游戏版号
    + 关税的增加，2018 中美贸易战下的外贸公司
    + 例如: 2008经纪危机, 希腊, 俄罗斯石油行业
+ 控制好风险，顺势额为，在趋势面前，最优秀的团队也可能无计可施    
- 财务风险
	+ 流动性风险
		+ current ratio = current asset/current liability
		+ quick ratio = (cash  + markable securities + receivables) / current liabilities 
		+ cash ratio = (cash  + markable securities) / current liabilities 
		+ ratio > 1, 三个指标逐渐严格，看公司是否有短期还债的能力
	+ 总债务风险
	    + 债务水平
	        - 四个个指标越小越好
            - debit to asset = total debt/total asset
            - debt to equity = total debt/total equity
            - debt to capital = total debt /(total debt + total shareholder's equity)
            - Financial leverage = average total asset / average total equity, average指一年之内
        + 偿还能力
            - interest coverage = earning before interest and taxtes / interest payment, 数值越大越好
            - fixed charge coverage =  earning before interest and taxtes + lease payment / interest payment, 数值越大越好
            - debt payment = total debt / net earnings, 小于4
        + 依靠债务提高收益率
        + 依靠债务维持竞争力
        + 依靠债务续命
	+ 利润率风险
	    + 比较同行业公司和竞争对手情况
	    + 比较和公司过去
	    + cross profit margin = gross profit / revenue, 成本控制，比对手高
	    + operating margin = earning before interest and tax / revenue， 运营能力，比对手高
	    + pretax margin = earnings before tax / revenue， 比对手高
	    + net profit margin = net earning / revenue, 比对手高
	    + return on asset = net earnings / average total asset, 比对手高
	    + return on equity = net earnings / average total equity, 大于15%
	+ 运营风险
	    + receivables turnover = annual sales / average receivable, 一年卖多少轮货，比对手高
	    + days of sales outstanding = 365 / receivables turnover, 比对手低
	    + inventory turnover = cost of goods sold / average inventory, 一年买多少轮货，比对手高
	    + days of inventory on hand = 365 / inventory turnover, 比对手低
	    + payable turnover = purchases / average trade payables, 比对手低，欠供货商的钱
	    + number of days payable = 365 / payable turnover, 比对手高，付款期限，越高越好
	    + cash convention cycle = days of sales outstanding + days of inventory on hand - number of days payable, 比对手低，比自己过去更低
- 数量分析 vs 质量分析
    + 数量分析1家公司是否过关，值不值得花更多时间研究
    + 数量分析是质量分析的数据化，帮助理解和分析公司，不要过于依赖和过度解读
    + NCAV = cash & cash equivalent + partial receivable + partial inventory - total liability 来计算安全边际
- 管理层风险
    + 最高决策人换人， 例如台积电张忠谋，王石，乔布斯
    + 团队分裂， 内讧，党争
    + 资本控制
- 市场风险
    + 利息大幅提高：反相关
    + 流动性不够：流动盘总量，平均每天交易量大小
    + 其他资产吸收资金
- 顾客风险
    + 需求变化， 电脑到手机，
    + 行为认知变化
    + 行为方式
    

## 买卖
#### 买卖的基础：目标的设定
+ 获得财富自由的主要方式，次要方式
+ 增加额外收入的方式
+ 培养一个职业技能，庄行进入资产管理行业
+ 培养一个爱好，感受投资的乐趣

#### 买卖的准备
+ 心态的建设
    + 有耐心
    + 不从众：不被市场情绪，不被价格走势影响。以事实和逻辑为判断标准
    + 只投资在目标价格之下的公司
+ 确定资金时长
    + 使用和积累资金分离
    + 长短期投资资金分离
    + 积累的投资资金和消费资金，应急资金分离
+ 建立选股标准
    + 必要标准: 公司实现盈利(业务盈利但是费用到运营或者研发)，有很强相对竞争优势或者已经形成护城河， 公司管理层可以信任
    + 非必要考量标准: 长期盈利是增长趋势，数量分析财报，质量分析公司
+ 建立股票池
    + 股票池股票数量， 100家公司长期追踪
    + 相对理想状态，15家懂的优质公司

#### 买入逻辑
+ 不懂不投，好股好价
+ 自己懂的公司
+ 优质的公司
+ 达到目标价格

#### 卖出逻辑
+ 发现自己判断错误
+ 公司质量变坏
+ 环境发生变化，不能判断未来长期利润
+ 提前收获了足够利润，冰倩有其他买入机会

#### 对比
+ 投资潜在获利对比
    + 投资潜在获利对比
    + 获利确定性的对比
+ 选择买入公司
    + 首先确定最优买入选择仓位
+ 买卖的方法
    + 定点买入：安全边际高，操作简单，可能会损失一些投资机会
    + 逐步建仓：安全边际相对不搞，抓住更多机会，操作复杂，承担的风险更高

#### 常见心态问题
+ 承压抛售
    + 买入正确
    + 市场价格下跌，心态收影响，感受资产在缩水
    + 受不了压力抛售
    + 症结
        + 没有真正懂公司，不确定公司长期盈利能力
        + 受不了压力而抛售
+ 过度恐惧
    + 判断正确，目标正确
    + 被价格下跌吓住，不敢买入
    + 症结
        + 没有真正懂得公司，不确定公司的长期盈利能力。
        + 不确定整体经济环境，对公司未来盈利的影响。
        + 不理解股票价格，短期内是被买卖双方的资金量决定的。
+ 放松标准
    + 建立的投资体系正确
    + 在价格飞涨下，放宽投资体系标准，去迎合更高的价格，从而可以投资。
    + 症结
        + 急于抓住投资机会盈利
        + 变动投资逻辑时，没有基于事实和逻辑，而是基于抢购的欲望
+ 频繁买卖
    + 公司选取正确，目标价格正确
    + 急于在价格上涨一些后获利了结，在价格还没有到达目标价位时买入。
    + 症结
        + 注意力焦点在价格波动上，在分析价格未来走向上
        + 没有按投资系统进行投资

#### 买卖的判断
+ 研究懂公司
+ 积累自己的股票池
+ 根据长期利润的确定性，和增长速度，在合适的价格/PE下买入。 
+ 在卖出逻辑下，卖出股票
+ 或长期持有

#### 买入
+ 无增长公司
    + 长期持有，盈利潜力有限 
    + 确认公司的长期盈利能力
    + 尽量在10倍PE以内买入
    + 举例:苹果
    + 安全边际比有增长的同等质量的公司更大 有明显催化剂，或分红水平满意，或股票回购水平满意
+ 稳健增长公司
    + 长期持有，盈利潜力很不错
    + 确认公司的长期盈利能力
    + 尽量在15倍PE以内买入
+ 快速增长公司
    + 长期持有，盈利潜力非常可观
    + 确认公司的长期盈利能力
    + 尽量在25倍PE以内买入
    + 特殊经济环境下的买入:
    + 全球整体经济萧条/衰退 ● 地区性、行业性经济衰退 ● 政策风险下
    + 买入的考量:对公司长期盈利能力的影响，公司有无破产风险。
    + 举例:巴菲特在08年金融危机下买入银行股和银行的可转债。

#### 卖出
+ 卖出:卖出逻辑
    + 发现自己判断错误
    + 公司质量变坏
    + 环境发生变化，不能判断未来长期利润
    + 提前收获了足够利润，并且有其他的买入机会
    + 理想状态
        + 无增长公司，获利50%-100% (持有年限不确定)
        + 稳健增长公司，年化获利> 10%  
        + 快速增长公司，年化获利> 20%

#### 长期持有
+ 增加获利的确定性，获得巨大财富的最有力帮手 ● 长期有成长性的公司
+ 需要持续跟踪公司和行业
+ 适用于:
    + 有持续回购的无增长公司
    + 稳健增长型公司
    + 快速增长型公司

#### 股票的持有
+ 买卖的目标：持有优质的公司
+ 持有的前提： 懂的优质的公司

#### 持有无增长型公司
+ 例如主要来源
	+ 市场的误判
	+ 公司的盈利 （分红和回购）
	+ 公司的长期提升
+ 适合买入：10PE一下
+ 7PE 买入，分红率8%
+ retained earning 可以持续产生一定投资回报
+ 长期持有，10%以上的回报

#### 持有稳健增长型公司
+ 例如主要来源
	+ 市场的误判
	+ 公司的盈利 （分红和回购）
	+ 公司的长期提升
+ 适合买入：15PE一下
+ 12PE 买入，分红率4%
+ retained earning 可以持续产生一定投资回报
+ 长期持有，16%以上的回报

#### 持有快速增长型公司
+ 例如主要来源
	+ 市场的误判
	+ 公司的盈利 （分红和回购）
	+ 公司的长期提升
+ 适合买入：25PE一下
+ 20PE 买入，增长率20%， 无分红
+ retained earning 可以持续投资回报
+ 长期持有，22%以上的回报

#### 懂得优质公司
+ Can it win
	+ 现在
		+ 是不是懂得相对竞争优势
		+ 有没有护城河
		+ 护城河是哪些
	+ 未来 （内部）
		+ 懂得多久可以持续长期保持护城河
		+ 管理层是否可以继续加深护城河
	+ 未来 （外部）
		+ 懂得潜在竞争对手
		+ 他们可能对目标公司形成挑战
		+ 3-5年内多大实现可能
		+ 对它产生什么影响
+ Can it earn
	+ 懂得3-5年公司利润
	+ 未来发展（内部）
		+ 公司最高决策者
		+ 管理层情况会怎么发展
		+ 公司文化怎么样发展
		+ 公司经营效率
		+ 公司产期利润有什么影响
	+ 未来发展（外部）
		+ 公司所在行业和外部环境影响
		+ 会对公司产期利润产生什么影响
	+ Is it cheap
		+  公司价值
			+ 是不是懂得它的价值
			+ 他的相对保守估计的价值是多少
			+ 3-5年后，10年后公司价值是多少
		+ 安全边际
			+ 现在价位给多大安全边际
			+ 最欢的情况，我损失会是多少
		+ 投资目标
			+ 我懂我的投资目标
			+ 能不能给所需要的回报
			+ 能不能满足投资需求

## 投资组合
#### 仓位和持股数量
- 没有唯一的标准
    + 开始的时候，以模仿为主。不要太分散，或太集中。
    + 努力建立自己的能力圈和优质股票池。
    + 达到一定理解的深度后，可以根据个人特点和目标，应用事实与逻辑，进行调整。
- 3个维度的考量
    + 潜在回报:长期的价值增长能力。增长速度、增长时间长度的综合考量。 
    + 确定性:公司的护城河，行业的特点，理解的深度。
    + 风险控制:市场提供的安全边际。
- 10%-25%之间
    + 真正弄懂:没有把握建仓10%，很多时候是还没有真正弄懂公司。没弄懂，就继续研究。
    + 集中投资:避免过度分散化投资。尤其是个人，因为看不过来。投资一堆不懂的公司风险更大。
    + 敬畏市场:即便把握很大，风险很小，很想把全部资金都投进去，也尽量不要超过25%。
#### 持股数量
- 一般4-10支股票之间(对应10%-25%仓位) 
- 太集中，则容易因一个公司带来严重损
- 太分散，则容易每一个公司都没有弄懂
#### 现金仓位
- 刚开始投资时，会比较高。
- 调整到比较好的长期持有状态下时，一般在0%-20%之间。
- 在市场太过高估，并且有很强经济环境风险时，可以升高现金仓位。
#### 现金仓位的价值:等待 现金仓位升高的几种常见情况:
- 没有好的投资机会，又不断收到分红;
- 投资的公司质量变差，卖出;
- 卖出提前获利的公司，正准备投资下一个优质公司。
- 整体经济环境变坏，价格太过高估，提升现金仓位进行抵御。
#### 功能
- 增长:更高的财富增长速度，需要更多的跟踪。举例:台积电。
- 现金流(分红):提供额外投资的现金，提供生活开销的使用。
- 防御:在经济大环境变差的情况下，维持总资产的产生现金流能力，或购买被低估公司的
- 划分仓位给不同功能，2种不同方式:
    + 固定比例给固定功能。例子，30%仓位增长，70%仓位分红。
    + 没有固定比例给固定功能。只看投资机会的相对风险和相对收益，决定仓
#### 选股
- 在能力圈范围内
    + 判断公司质量的较高准确率的范围 
    + 不代表判断就没问题
    + 以事实和逻辑为判断标准
    + 不以别人甚至权威的观点为标准
- 是优质公司
- 满足投资目标
- 拥有足够的安全边际
- 最少确认方面:
    + 相对竞争优势/护城河，未来利润可期
    + 最高决策者的经历和决策倾向性
    + 主要管理层有足够能力，公司文化健康
    + 公司总体战略，竞争战略，资源战略
    + 所在行业现在体量和未来发展
    + 没有重要的财务风险
#### 仓位调整
- 仓位调整的前提
    + 知道优质公司中，投资机会的相对排序
    + 投资最好的那些机会:能力圈和股票池范围的重要性。 
    + 如果没有办法排序，很可能还没有真正懂得公司。
- 仓位调整的常见情况
    + 公司质量发生变化，未来长期盈利能力发生变化。
    + 又发现了新的优质公司，或之前没有机会买入的优质公司进入可以买入的价格区间。投资排序的 顺序发生变化。
    + 整体经济情况发生变化。
- 仓位调整后的追踪和验证
    + 追踪内容:公司的质量，盈利，变化等等。
    + 验证:投资机会的排序逻辑，根据收集到的事实和逻辑，是否成立，并持续保持正确。
- 总结
    + 仓位和持股数量:判断正确时，有效盈利。
    + 功能:明白原因，清楚判断标准。
    + 选股:确认什么该投，什么不该投。
    + 仓位调整:持续保持，合理的投资状态。



## 投资案列

#### 经典正面案例:American Express 

+ 买入机会:公司危机
+ 持有时长:3-4年
+ 获得利润:154% 
+ 最高时达到仓位:40% 经验:
+ 一定要做足研究公司，真正理解公司。
+ 遇到把握很大，确定性很高的投资机会，放足仓位。 
+ 值得长期持有的公司，不要过早卖掉。

#### 经典正面案例:GEICO

- 买入机会:公司危机
- 持有时长:44年以上
- 获得利润:More than 10 Billion
- 成本:45.7M(51%)+2.3B(49% after 20 years)
- 如果足够幸运，有些公司不需要卖出:永久仓位
- 保险公司的float对于投资的额外助力。
- 有的钱，公司不该赚。有的利润，投资者不该赚。

#### 经典正面案例:See’s Candies

- 买入机会:管理层寻找退出机制(公司是质量公司，品牌优秀，管理者想要退休，需求接

  盘人，巴菲特买下，从公司收获了很好的现金流。)

- 持有时长:48年以上

- 获得利润:2 Billion

- 成本:25 Million

- 长期优秀质量，形成强大品牌的价值。

- 强现金流公司，对投资的帮助:永久仓位

- 优秀的管理层，很多时候不是为钱工作，而是为了工作而工作。 

- 一个区域做得很成功，扩张到另一个区域未必成功。

#### 经典正面案例: Wesco Financial

- 买入机会:逐步增加股权
- 持有时长:46年以上
- 获得利润:>2 Billion
- 成本:30 Million 经验:
- 换了管理层，公司就不是同一个公司。
- 在确定永久仓位中，管理层的key person是至关重要的。 
- 管理层的主要人事变动，需要持续跟踪。

#### 反面案例:Berkshire Hathaway

 ● 错误原因:为了出气
 ● 主要损失:早期的回报率，和大量的时间和精力
 ● 做理智的投资者，投资行为不要被情绪左右。
 ● 每个人投资都会犯错。如果错了，不要对自己太过严苛，记录教训以后少犯就好。 ● 持续做投资比每次都正确重要得多。

#### 反面案例:Cities Services

 ● 错误原因:过早地卖出 ● 获得利润:4.5%
 ● 主要损失:400%的回报
 ● 买入时，如果只是为了获利10%, 就不要买了。 ● 买真正懂得的公司，做真正值得的投资。

#### 反面案例:IBM

- 错误原因:误判了能力圈范围
- 获得利润:48% in 10 years
- 主要损失:错过了真正拥有竞争优势的Microsoft
- 不要怕加错仓:它一定会在某个时间发生的。
- 投资策略相对保守，有时投资错了，10年收货48%回报，也并没有亏。
- 卖出的逻辑，不是挣没挣到钱，而是符合不符合投资标准，判断的底层逻辑是否正确。

#### 常见错误: 

##### 能力圈外

+ 在投资前，不要盯着可能的利润，盯着判断的准确度、获利的概率。 
+ 每次投资之前，问自己:我懂不懂这家公司?
+ 跟其他价值投资者讲解自己的投资逻辑，帮忙查看投机漏洞。
+ 投资之后，持续跟踪研究，发现判断失误，及时卖出仓位
+ 持续研究公司，扩大能力圈范围。
+ 持续跟其他价值投资者交流，扩大能力圈范围
+ 组建所研究行业的微信群，聚拢行业内人士。
+ 发现优质公司，放入研究列表，不要忘记。
+ 养成阅读年报的习惯。
+ 养成阅读经典价投书籍的习惯，经典书反复看。

##### 能力强内优质公司，有长期价值，过早卖出仓位

+ 盯着长期价值，不要盯着短期波动。
+ 卖出时找其他价值投资者，述说自己的卖出逻辑，帮忙查看逻辑漏洞。
+ 养成长期持有优质资产的习惯。
+ 经常查看巴菲特和其他价值投资者长期持有的获利情况，使注意力放在长期持有获利上。
+ 在生活中习惯，使用事实和逻辑进行推理，判断和决策。 
+ 积累生活中、投资中因独立思考能力而判断成功的案例。 
+ 养成集中投资的习惯
+ 拒绝买入质量不是非常高的公司。

##### 能力圈内，投资过度分散，研究时间不够，导致准确率下降

+ 拒绝买入，质量不是非常高的公司。
+ 拒绝买入，达不到安全边际目标的公司。
+ 养成习惯，有极大把握才投资。不然不投，耐心等待。
+ 每周拿出固定的至少3-5小时，进行公司研究和持续追踪。
+ 分配专门的时间给主要仓位公司，进行持续追踪和研究。
+ 定期和其他价值投资者交流，讲述自己新的研究成果。
+ 设置手机提醒，到了固定的时间，就去研究主要仓位的公司
+ 没有完成主要仓位公司的研究时间，不去研究别的公司。

##### 没有持续研究更多公司，扩大能力圈，仓位无法变动。

- 组织或参加持续研究公司的小组，定期互相更新研究进展。
- 分配专门的时间给研究新的公司。
- 分配专门的时间跟新研究公司的所在行业人员见面/电话请教行业知识。
- 把自己研究新公司设定的目标和计划，告诉自己重要的朋友，并让朋友帮忙监督和查看进度。

##### 换手率太高，频繁变动仓位。

+ 每次买卖之前，检查自己的买卖逻辑，并跟其他价值投资者述说逻辑，帮忙检查。
+ 买入时，清楚地告诉自己，目标是长期持有
+ 因市场波动而情绪受到影响时，及时跟其他价值投资者交流。



## 常见情况

#### 额外资金的投资

+ 工资或额外收入的储蓄部分
+ 股票投资分红
+ 其他投资的收益

+ 应对:符合标准才投资

- 无投资机会:升高现金仓位(不要担心持有现金)
- 有投资机会:投资到第一投资标的上(拥有对投资标的的排序能力)，第一投资标的达到
- 仓位目标，则投资到第二投资标的

#### 市场非理性下跌

- 确认 holding power 没有问题，持有仓位不用卖出 
- 如果有额外资金，在目标价位买入投资标的

#### 市场非理性上涨

+ 有其他优质的投资机会:可以考虑获利卖出
+ 没有其他优质的投资机会:继续持有
+ 经济风险很大，但价格非理性上涨:可以考虑减仓，升高现金仓位

#### 整体经济出现问题后的波动

+ 应对:盯住公司长期盈利水平
+ 以公司未来长期盈利水平为判断依据，抵抗价格波动影响
+ 如果市场非理性上涨，可以考虑升高现金仓位
+ 如果公司未来长期盈利能力遭到严重打击，甚至有破产风险，卖出

##### 需要使用一部分资金 应对:保护投资目标

- 从现金仓位调资金
- 无现金仓位，优先从辅助功能仓位调资金
- 无辅助功能仓位，优先从主要功能中长期贡献更少的仓位调资金
- 投资目标:财富自由后的现金流
- 主要功能:分红类公司
- 辅助功能:成长类公司
- 没有现金仓位时，优先卖出成长类公司，调出资金

#### 判断失误

+ 不要试图用错误的方法赚钱
+ 直接卖出仓位
+ 不要等待，如果有损失，则是判断失误的代价

#### 出现更好的投资机会 应对:新投资机会需要更高标准

+ 如果有50%以上更好的长期回报，可以考虑换仓
+ 如果当前仓位价格预支了未来很多年的利润，可以考虑换仓 ● 其他情况，继续保持现有仓位

#### 一段时间后的仓位不平衡 应对:盯住公司质量

+ 汇报更高的公司，仓位可能超出25%，不用减仓，继续持有
+ 汇报更低的公司，仓位可能低过10%，不用加仓，继续持有
+ 现金仓位可能因为没有好的投资机会，变得很高，不要着急投出，继续持有现金仓位

#### 一段时间后的公司质量变化 应对:盯住公司质量

+ 如果质量达不到标准，卖出仓位
+ 如果质量依然达标，但达不到高仓位的标准，可以减仓 ● 如果质量提升，超出了现有仓位的质量水平，可以加仓

## 复盘与升级

+ 记录与总结记录下思考和决策的过程
+ 提高与升级:通过事实反馈提升理解和实践能力

复盘是对自己思维过程的觉知, 记录和纠正。

事实和假设:投资时的重要事实和假设

+ 事实是发生的真实情况的具体反映
+ 假设是对无法确定的事的假定。

### 记录和总结

例子:Facebook用户

+ 事实:Facebook是目前social media里用户人数最大的
+ 假设:Facebook里统计的用户活跃度是合理的统计方法。

逻辑链条:投资时的重要逻辑链条

+ 逻辑链条是由事实和假设推导的。
+ 逻辑链条有缺失或错误，会导致判断错误，决策错误。
+ 真实记录自己逻辑推理过程中的错误，那你就是超牛的人，而不是犯了错的人。

例子:错误的逻辑链条

+ 过去A公司股价一直没有跌超过20%过
+ 所以A公司的股价很稳定，如果有下跌15%就已经是很好的买入机会。
+  Missing link: 过去不等于未来。

判断:投资时对未来的预期

+ 判断包括:行业洞见，未来的盈利，保持护城河，未来的成长速度等等
+ 判断里有定性判断，也有定量判断。
+ 准确记录主要的判断依据。

举例:2018年底买入脸书判断

- Social Media行业里，其他公司不具备跟脸书形成真正竞争的能力。
- 脸书目前的最主要的竞争对手是谷歌，它们各自在展示位广告，搜索广告和视频广告中拥有独特的护城河。
- 脸书拥有的提价能力，和进一步蚕食传统广告市场，可以支撑起15%甚至20%的年化利润增长。
- 脸书当时的价位，提供了满足要求的安全边际。

决策:投资时的重要决定

+ 基于判断和投资体系，应该怎样决策
+  实际的决策是怎样的。

差异:差异在哪里，差异的来源是什么

+ 应该的决策和实际的决策之间，有没有差异。
+ 差异的来源是什么。

总结:哪些需要保留，哪些需要防范

+ 保留的，进一步明确，或者加入到投资体系中。

+ 防范的，在需要的时间点，对自己进行提醒。



### 提高与升级

去除错误沉淀经验:假设、逻辑链条、判断、决策

假设

+ 是不是对自己思维的觉知不够?更多觉知思维的练习
+ 是不是没有意识到自己有做假设?更多问自己有什么假设的练习
+ 还有什么其它的假设错误?在哪个节点提醒自己不要犯类似的错误? ● 有哪些假设做得很好?

逻辑链条:

+ 是不是有一些已有的有问题的思维方式
+ 这些思维方式有问题的点，到底在哪里
+ 我可以用什么样的思维方式替换它?
+ 沉淀下来哪些逻辑链条?

判断:

+ 判断时，脱离逻辑链条的点在哪里
+ 判断时，做得非常好的经验是什么?

决策:

+ 决策时背离判断的点在哪里
+ 决策时，做得特别好的地方在哪里?

#### 能力圈范围升级:更准确的划分能力圈范围

+ 能力圈的准确程度，远比能力圈大小重要。去掉“伪能力圈”。
+ 用新的经验，增大能力强范围。

#### 交易升级

##### 买卖

+ 记录自己买卖时犯过的错误，时刻提醒自己。
+ 这些错误，也是未来自己最容易犯的错误。 

##### 仓位

- 2个判断能力完全相同的人A和B，投资目标一样。
- 对25家公司有相对准确的判断力。
- A投资了6家最好的公司，分配仓位25%、20%、15%、15%、10%、10%，另5%现金，长期持有。
- B投资了25家公司，每家公司4%，无现金仓位，并且频繁交易买卖。

##### 练习

+ 频繁交易，有很多钱之外的成本:时间、精力、注意力成本。
+ 还有一个很大的代价:因为心态不稳，影响到其它方面。
+ 记录自己付出过的成本和代价，下次再想频繁交易时，看一看自己记录的，还有跟其它价值投资者寻求帮助。

#### 心态升级:减少心态对判断和决策的影响

+ 影响判断:本来可以判断正确，被心态影响而判断错误。
+ 影响决策:判断已经正确，但决策时并没有依据判断做决策。



## 实践方法

### 实践的必要性

+ 不实践，没办法内化学到的知识 
+ 不实践，过一段时间就会忘记了
+ 不实践，没有办法有真实的利润

### 价值投资实践的3个关键点

+ 读书
  + 读最经典的价值投资书籍，每天最少读一页。
  + 读作者是实战有结果的，理论派的可以看，但不要模仿。类比例子:烧陶瓷。 ● 最经典的书籍，要反复读。
+ 分析公司
  + 读年报:逃避不了的，一定要读。
  + 跟行业内专业人士请教:真正理解行业和公司。
  + 建立行业人士的微信群。
+ 做投资并复盘
  +  使用真实资金投资。
  + 资金量不能过大，威胁到财务安全。
  + 资金量不能太小，亏损后没有感觉。
  + 经常跟其他价值投资者讨论:检查自己和扩大能力圈。
  + 开始时，最重要的是，达到真正内化的懂。
  + 先模仿再调整。

### 实践中最大的困难

+ 没有坚持下去:设定有动力的目标，跟其他人一起，跟随一个小组。
+ 害怕失败/过分自责:失败是正常的，是一个结果。结果失败，不等于人失败
+ 太过乐观:好结果后容易过度乐观，降低投入或者反思。

### 这周内要完成的3件事

+ 制定一个让自己非常有动力的目标。
+ 制定每日行动计划:看书、分析公司、实践。
+ 制定监督机制:其他价值投资者，支持的朋友或家人。

## 市场特点

### 多市场投资

+ 避开某个整体市场的高估价格
+ 收获更多投资的机会
  + 总体股票数量
  + 不同时期的各个市场平均高低水平不同
  + 经济危机，行业危机，个股危机更多机会

+ 帮助选择更适合自己的投资市场
  + 大体量市场，波动更大
  + 小体量市场，波动更小

+ 五个市场
  + 美国市场
  + 新加坡市场
  + 香港市场
  + 中国市场
  + 欧洲市场

### 美国市场

+ 资金的机构化程度高
  + 美国的资产管理行业极度发达
  + 养老金，政府投资资金，巨富家族资金都会投入很大部分到美国股市

+ 市场波动较大
  + 机构主力资金的规则:相对回报率竞赛
  + 看好时，会整体追着跟进;不看好时，会整体争相逃出

+ 市场被预期左右严重
  + 资产管理机构的游戏规则:预期游戏
  + 股票的价格，更多的时候存在于整体预期中，而不是公司价值中
  + 一旦大大超出预期，或者远远不及预期，就会引发价格大幅波动

+ 个股经常估值过高，或者过低
  + 预期会大部分趋同，或取平均
  + 大额资金方向也会相同
  + 会有很多买入机会

+ 监管更多细节，会计准则更严苛

  + 华尔街和监管机构长期博弈

  + 在很多轮的迭代中，更多规则被确定

  + 但是每次华尔街都可以同一个新办法，再次制造泡沫和问题，所以不要去人多的地方投资

    

### 新加坡市场特点

+ 成长受限
  + 市场纵深太小，扩张需要出国
  + 产品和服务，不同国家需求不同
  + 资源积累不够，出国风险较大

+ 波动偏小
  + 总体资本量级不大，波动较小
  + 本土市场格局相对稳定

+ 分红偏高

+ REITs有些质量不错
  + 新加坡REITs相对规范
  + 考察起来相对容易
  + 有一些资产质量不错

+ 相对竞争和风险偏小
  + 市场体量相对较小，优秀团队较相对竞争偏小
  + 优质公司的质量变坏概率较低，风险偏低



### 香港市场特点

+ 比较相信实实在在的业绩和利润
  + 业绩和利润没有展现的时候，不相信公司价值
  + 描绘前景，很多时候没有用
  + 新股要慎入

+ 背靠大陆，有很多大陆的优质公司在香港上市，未来会有更多
  + 中美矛盾会让更多公司最后在香港上市;
  + 在美国的也会迁到香港很大一部分

+ 是外资可以自由投资中国优质资产的地方
  + 资本是追逐利润的，外资投资中国，通过香港是最佳的渠道



### 中国市场特点

+ 政府有绝对的影响力
  + 中国的各个行业，受政策影响非常大
  + 政府对行业的政策会影响利润
  + 投资者对政府政策的导向，非常相信

+ 政府的资金量级也很大

+ 整个市场未来会有越来越多的好公司
  + 中国的整体经济体量能支撑更多的产品和服务
  + 中国人骨子里有不断向上的品质



### 欧洲市场特点

+ 有很多老牌优质公司;很多公司有自己的投资部门，收益甚至比主要业务收益高
+ 分析员相对匮乏，中小型公司，有很多好的投资机会，甚至大公司



## 市场细节

### 美国市场

#### 交税

+ 资本利得税(Capital Gain Tax)
  + Dividend: 30% tax(美国公司)
  + Dividend:0(非美国公司)
  + Capital Gain: 0 
+ 如果投资目标主要是收获现金流，要特别小心 

#### LOT SIZE

+ 各种 Lot size 都有
+ 一般比较贵的，可能 lot size 也只有一股(Berkshire，Google etc)
+ 购买之前，要先查清楚 lot size

####  交易限制

+ T+0:当天买入的股票，当天可以卖出
+ 熔断:股指在一天内下跌 7%，13%，20%(标准为 S&P500 实数)
  + 不要做 Day Trading
  + 如果万一出错，放错了股票代码，可以当天卖掉

####  代持和费用

+ 一般需要银行的 Custodian Account 代持股票
+ 会有 Custodian Fee，一般是按股票数，或者金额来收费，不同平 台不一样
+ 一般金额不大。要小心，有些平台的 trading account 里如果有现金，也会触发额外 的 custodian fee

#### 交易时间

+ 纽交所:9:30am to 4pm
+ 美交所:9:30am to 4pm
+ 纳斯达克:9:30am-4pm



### 香港市场

  + 不要做 Day Trading
  + 万一出错，放错了股票代码，可以当天卖掉

#### 代持和费用

+ 一般需要银行的 Custodian account 代持股票
+ 会有 Custodian Fee，一般是按股票数，或者金额来收费，不同平 台不一样

## 总结

### 能力圈

+ 准确判断自己额能力圈，是投资中安身立命的根本
+ 时不时问自己:这个公司，我真的懂么?如果不懂，在什么方面?

#### 安全边际

+ 投资优质的公司，一样可以亏钱
+ 在投资一支股票前，问自己:我留出了多大的安全边际?都在哪 些地方?

#### 护城河

+ 护城河是公司保护自己利润的能力
+ 寻找有护城河的公司，收获长期利润
+ 找到这样的公司，逐步做到比绝大部分人更懂这家公司

#### 价投三原则总结

不懂不投，好股好价

#### 公司财务分析 3 个表

+ 损益表:公司盈利情况
+ 资产负债表:公司财务情况
+ 现金流量表:公司现金流情况

#### 公司财务分析

+ 记住常用名词的含义
+ 真正得去读年报表，开始时会慢，后来会越来越快
+ 重点要抓住反映公司真实情况的数据，和反常的数据
+ 留心没有被披露或解释的情况

#### 市盈率(PE Ratio)

+ 是我们判断公司价位的最主要方式
+ 相比现金流折现更容易操作，误差更小 
+ 最近一年的盈利，不能代表长期平均利润时，用长期平均利润计 算市盈率
+ 没有一个标准的买入公式，要根据具体的公司质量来决定

#### 业绩趋势

+ 公司需要长期有增长，才有价值提升。怒让就是烟蒂投资法，持 有时间越长，风险越大，回报越低
+ 长期的增长，也提供额外的安全边际
+ 过去的业绩趋势，有一定指导作用。但仍应该由现在真实的情况 决定。

#### 利润率

+ 较高的利润率，可能是护城河的表现
+ 投资较高利润率的公司，风险也会更小。同样是净利润下降 7%， 从 20%或者 3%的净利润率下降，结果是完全不容的
+ 尽量不要投资还没盈利的公司

#### 负债率

+ 盈利能力很强的公司，不需要负债
+ 适度使用负债，可以增加 ROE，增多回报
+ 不要投资过度使用负债的公司，即便现在看起来没事，长期来看， 黑天鹅一定会来

#### 股本回报率(ROE)

+ ROE 的高低，影响公司长期的价值提升
+ 考察 ROE 时，要结合公司的负债率。单纯的使用债务减少运营 中需要的资金，推高 ROE 的方式不可取

#### 3 分钟筛选法

+ 用营业额趋势，ROE, Net Margin, Debt to Asset,可以迅速对目标 公司的质量有一定了解
+ 不符合某一项，是优质公司的概率会降低很多，而不是肯定不是 优质公司。

#### 公司战略

+ 公司的总体战略，竞争战略，资源战略，是帮助理解一个公司， 很有效的角度
+ 也是预测一个公司未来发展和利润的有效数据
+ 理解公司，行业，战略，可以去跟行业内专业人士请教行业的知 识。形成对真实情况的理解。

#### 管理层

+ 对最高决策者，性格决策倾向性的理解，可以帮助更好判断公司 战略
+ 中层管理者能力到位，会确保公司战略，可以得到实现

#### 行业

+ 行业的体量，未来的发展，可以对公司利润产生巨大影响
+ 收集行业数据时，也要同时结合我们对行业的理解，判断数据是 否合理
+ 了解行业属性，帮助避开不合适的公司
+ 研究行业上下游，发现更多的优质公司，并对行业本身运行规律 加深理解

#### 竞争格局

+ 垄断格局是最理想的长期投资格局
+ 寡头竞争，差异化的完全竞争格局也都是不错的长期投资格局
+ 研究现在的竞争格局，以及未来大概率形成的竞争格局，帮助对 公司未来发展形成判断

#### 风险

+ 认知风险，是最大的风险
+ 心态风险，是最难克服的风险
+ 投资前要理解竞争风险，行业风险，政策风险，经济环境风险， 财务风险，运营风险等，留出合理的安全边际
+ 投资后要持续追踪公司，在风险过大，影响到公司长期盈利能力 时，可以考虑卖出或减仓

#### 买卖

+ 遵守买入和卖出的逻辑，避免受贪婪或恐惧情绪影响做出不理智 的决定
+ 进行买入卖出的目标，是为了持有优质公司
+ 长期持有优质公司，实现利润 买卖的总结:买卖少出错，持有获利润

#### 投资组合

+ 注意仓位分配，和持股数量，做到相对集中的合理分散
+ 不要忽视现金仓位的在等待时创造的价值
+ 在做投资时，理解每个公司在投资组合中起到的功能是什么
+ 根据投资目标，进行投资 5.避开常见的错误，了解常见情况下的应对方法

#### 投资跟踪与复盘

+ 每个人容易犯错的点，都是相对固定的
+ 找出来这些点，并且尽量少犯
+ 每个人的优势也是独特的，找到并更多使用 4.练习自我思维的觉察，提升思考质量

#### 市场特点

+ 多市场投资具有优势
+ 理解不同市场的特点，并让它为自己的投资服务
+ 了解市场的一些细节
